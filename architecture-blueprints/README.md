# High-Level Architecture Blueprints
System design diagrams for Multi-Cloud and Hybrid solutions.

## Methodologies & Standards
* **TOGAF:** Enterprise Architecture development method for aligning IT with business goals.
* **C4 Model:** Hierarchical approach (Context, Containers, Components, Code) for visualizing software architecture.
* **Well-Architected Frameworks:** Adherence to AWS and Azure pillars (Operational Excellence, Security, Reliability, Performance Efficiency, and Cost Optimization).

## Tools
* Microsoft Visio, Draw.io, Lucidchart.
